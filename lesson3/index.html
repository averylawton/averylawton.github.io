Site description

The purpose of this document is to describe the weatherahead.com website. it will describe the purpose and intended audience, organization and structure, style elements, etc. It will be used as a planning tool for the personal site, weatherahead.com, to be developed this semester.

site name

This site is going to be called Weatherahead.com. It is a small home town's home page, focoused mainly on weather forcasts.

weatheraway.com's purpose

The purpose of this website is to give daily updates of a small town's weather, events, and pictures to advertize the town's attractions. This town is known for frequent tornadoes, so storm chasers are frequent tourists.

Target audience

The target audience of the weatheraway.com is: Local families, tourists, tornato enthusiasts and others interested in the weather of the town weatherahead is based in.
Age: 20+
Motivation: To Learn more about the weather and attractions of the town weatherahead is based in.
